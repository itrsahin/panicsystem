# Panic System Frontend

Bu klasor Panic System'in web arayuzunu icerir. Proje React, TypeScript ve Vite ile hazirlanmistir.

## Gelistirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Icerik

- `src/`: uygulama kodlari
- `public/`: sabit dosyalar
- `vite.config.ts`: Vite ayarlari
- `package.json`: script ve bagimliliklar

## Not

Bu frontend, ayni repodaki backend ile birlikte calisir. Genel proje dokumani kokteki `README.md` dosyasindadir.
